# Traffic-data-excel-dashboard
Overview The Traffic Data Dashboard provides insights into website performance using Excel pivot tables and raw traffic logs. It helps track visitor behavior, campaign effectiveness, and conversion performance across multiple channels.
Key Metrics
- Bounce Rate %
- Goal Conversion Rate %
- Visitor Count (New, Returning, Guest)
- Conversion % by Channel, Campaign, Page
- Avg. Session Duration
- Pages Per Visit
- Total Page Views
Data Sources
- Channels: Direct, Organic, Paid Search, Referral, Social, Email, Defect
- Campaigns: Campaign 1–4
- Pages: Page 1–3
- Visitor Types: New, Returning, Guest

Usage
- Open Traffic_Data_Dashboard.xlsx.
- Use the Dashboard sheet for a high-level view.
- Explore Pivot sheets for KPI breakdowns.
- Review the Data sheet for detailed daily logs.
Insights
- Direct traffic = high bounce rate, steady volume.
- Social traffic = strong engagement (long sessions, more pages).
- Email campaigns = moderate conversions, shorter sessions.
- Paid search = mixed results, campaign-dependent.
- Referral traffic = balanced conversions and engagement.
